Decentralised Federated Learning Framework for Traffic Optimisation in Smart Cities

Project Overview

This project addresses inefficiencies in traffic light synchronization that disrupt traffic flow and waste time. Current systems with intersection-based sensors fail to account for the broader flow of traffic between adjacent intersections. To resolve this, the proposed solution involves developing a decentralized system utilizing IoT devices within a dedicated network and a Federated Learning Framework.

The IoT devices will operate exclusively within their own secure network, collaborating only with each other and the learning algorithm. This ensures seamless peer-to-peer data sharing and processing without external interference. The system will optimize traffic flow using inflow-outflow equations, particularly effective in areas with one-way roads. By collecting real-time traffic data and continuously updating through decentralized learning over days, weeks, and months, the system will enable smart, adaptive synchronization of traffic lights. This approach enhances urban traffic management and minimizes congestion.
